# Visual JSON Editor

A Simple visual editor for a JSON object using React (with NextJS) and Codemirror.
NextJS Mainly used for easier deployment using vercel.

## Running Locally

```bash
npm install

# Either
npm run dev
# Or
npm run start
```

## Deploy

Check out the site live on [vercel](jsonvisual.vercel.app)
